## ✦ Что это такое

**KitsunePreset** — расширение для SillyTavern, которое добавляет плавающую плашку управления пресетами прямо в интерфейс чата. Больше не нужно вручную переключать пресеты при смене персонажа — расширение запоминает нужный пресет для каждого персонажа или чата и применяет его автоматически.

---

## ⚡ Возможности

| | |
|---|---|
| 🪟 **Плавающая плашка** | Компактный виджет в углу экрана, перетаскивается в любое место |
| 👤 **Привязка к персонажу** | Каждому персонажу — свой пресет |
| 💬 **Привязка к чату** | Отдельный пресет для конкретного чата (приоритет выше) |
| 🔄 **Авто-переключение** | Пресет меняется сам при смене чата или персонажа |
| 📌 **Режим докинга** | Плашка встраивается прямо в чат |
| 🔍 **Массовое назначение** | Назначай пресеты всем персонажам сразу из боковой панели |
| 🎨 **Интеграция с темой** | Берёт цвета из активной темы SillyTavern |

## 📦 Установка

### Способ 1 — через интерфейс ST (рекомендуется)

1. Открой SillyTavern → **Extensions** → **Install Extension**
2. Нажми **Install from ZIP**
3. Выбери файл `KitsunePreset_v8_fixed.zip`
4. Перезагрузи страницу

### Способ 2 — вручную

1. Распакуй архив в папку:
   ```
   SillyTavern/public/scripts/extensions/third-party/KitsunePreset/
   ```
2. Убедись, что в папке есть три файла: `manifest.json`, `index.js`, `style.css`
3. Перезагрузи SillyTavern

## 🎮 Как пользоваться

### Плавающая плашка

После загрузки в правом верхнем углу появится плашка с аватаром текущего персонажа.

```
┌─────────────────────────────┐
│  🦊  Персонаж  ✦            │  ← кликни для открытия панели
└─────────────────────────────┘
```

**Плашку можно:**
- Нажать — открыть панель управления
- Перетащить — переместить в любой угол
- В настройках — переключить в режим докинга (встраивается в чат)

---

### Панель управления

Открывается кликом по плашке. Содержит:

```
┌──────────────────────────────┐
│  [Аватар]  Имя персонажа     │  ← шапка, за неё тащить
│            ID чата           │   
├──────────────────────────────┤
│  ✦ Пресет   [Выбрать ▾]      │  ← текущий/выбрать пресет
├──────────────────────────────┤
│  Привязать: [💬 К чату]      │  ← привязать пресет
│             [👤 К персонажу] │
├──────────────────────────────┤
│  Все привязки                │
│  [💬 Чаты (3)] [👤 Персонажи]│  ← список всех привязок
├──────────────────────────────┤
│  ⚙ По умолчанию  [---▾]      │  ← fallback пресет
├──────────────────────────────┤
│  [📌 К чату]  [⚙]            │  ← докинг / настройки
└──────────────────────────────┘
```

---

### Привязка пресетов

#### К персонажу
1. Открой чат с нужным персонажем
2. В панели выбери пресет из дропдауна **✦ Пресет**
3. Нажми **👤 К персонажу**  
   → кнопка подсветится — привязка сохранена
4. При следующем входе в любой чат с этим персонажем пресет применится автоматически

> Нажать на уже активную кнопку — **снять привязку**

#### К чату
1. Открой нужный чат
2. Выбери пресет → нажми **💬 К чату**

> **Приоритет:** привязка к чату → привязка к персонажу → пресет по умолчанию

#### Пресет по умолчанию
Применяется для всех чатов **без** привязки. Настраивается в секции **⚙ По умолчанию**.

---

### Настройки (кнопка ⚙ в панели)

| Опция | Описание |
|---|---|
| **Авто-смена при переходе** | Применять привязанный пресет автоматически при смене чата |
| **Уведомления** | Показывать toast при применении пресета |
| **Цвет темы ST** | Использовать акцентный цвет из темы SillyTavern |
| **Включено** | Включить / отключить расширение |
| **⟳ Обновить пресеты** | Перечитать список пресетов из ST |

---

### Массовое назначение (боковая панель)

В боковой панели SillyTavern → Extensions → **✦ KitsunePreset**:

1. Используй поиск для фильтрации персонажей
2. Напротив каждого — дропдаун с пресетами
3. Нажми **✓** рядом с персонажем для сохранения
4. Или назначь пресет всем сразу, не открывая чаты

## 🔧 Отладка

### Плашка исчезла или вышла за экран
→ Нажми **🪟 Восстановить окно** в боковой панели расширения (Extensions → KitsunePreset)  
→ Плашка вернётся в правый верхний угол

### Пресет не применяется автоматически
1. Проверь, включена ли галочка **Авто-смена при переходе** (⚙ в панели)
2. Проверь, включена ли галочка **Включено**
3. Убедись, что привязка сохранена — кнопка привязки должна быть **подсвечена**
4. Если ST использует непривычный API-бэкенд — нажми **⟳ Обновить пресеты**

### Что-то сломалось
→ Нажми **⟳ Перезапустить** в боковой панели — мягкий перезапуск без перезагрузки страницы

### Логи в консоли браузера
Открой DevTools (`F12`) → вкладка **Console** → фильтр **`[✦KP]`**:
```
[✦KP] applying preset: MyPreset
[✦KP] bound char_5 → MyPreset
[✦KP] observer fired, lock=false
```

### Сброс всех настроек
- Кнопка **🗑 Удалить все привязки** в боковой панели — удаляет только привязки  
- Полный сброс через DevTools Console:
  ```js
  localStorage.removeItem('kp_kitsune-preset');
  location.reload();
  ```
